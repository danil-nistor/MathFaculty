## 1. Установка Node.js

### Windows

1.  Откройте браузер и перейдите на сайт: [https://nodejs.org](https://nodejs.org)
2.  Нажмите зеленую кнопку **LTS** (Recommended for most users).
3.  Скачается файл-установщик (например, `node-v20.x.x-x64.msi`).
4.  Запустите его и нажимайте **Next**, соглашаясь с настройками по умолчанию.
    *   ✅ Убедитесь, что стоит галочка **"Add to PATH"**.
    *   ✅ Убедитесь, что установлен **npm package manager**.
5.  После установки **перезапустите терминал** (Command Prompt или PowerShell).

### macOS

1.  Откройте браузер: [https://nodejs.org](https://nodejs.org)
2.  Скачайте версию для **macOS** (файл `.pkg`).
3.  Запустите установщик и следуйте инструкциям.
4.  После установки **перезапустите терминал**.

*Альтернатива для macOS (через Homebrew):*
```bash
brew install node
```

### Linux (Ubuntu/Debian)

Откройте терминал и выполните команды по очереди:

```bash
# Обновление списков пакетов
sudo apt update

# Установка curl (если нет)
sudo apt install -y curl

# Добавление репозитория Node.js 20 LTS
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -

# Установка Node.js
sudo apt install -y nodejs

# Проверка
node -v
npm -v
```

## 2. Создание рабочей папки

Откройте терминал (Command Prompt, PowerShell или Terminal) и выполните:

```bash
# Создаем папку для теста
mkdir mammoth-test

# Заходим в папку
cd mammoth-test

# Инициализируем проект (создаст package.json)
npm init -y
```

**Что произошло:**
*   Создана папка `mammoth-test`.
*   Внутри создан файл `package.json` (он хранит информацию о проекте).

---

## 3. Установка Mammoth.js

В той же папке выполните команду:

```bash
npm install mammoth
```

**Что произошло:**
*   Скачана библиотека `mammoth`.
*   Появилась папка `node_modules` (здесь лежат библиотеки).
*   Файл `package.json` обновился.

## 4. Создание скрипта конвертации

В папке `mammoth-test` создайте текстовый файл и назовите его `convert.js`. Либо используйте готовый из репозитория.

> ⚠️ **Важно:** Убедитесь, что расширение файла именно `.js`, а не `.txt`. В Windows может потребоваться включить отображение расширений файлов в проводнике.

Откройте `convert.js` в любом текстовом редакторе (Блокнот, VS Code, Notepad++) и вставьте код:

```javascript
const mammoth = require("mammoth");
const fs = require("fs");
const path = require("path");

// Пути к файлам
const inputFile = path.join(__dirname, "test.docx");
const outputFile = path.join(__dirname, "test.html");

// Проверка наличия входного файла
if (!fs.existsSync(inputFile)) {
    console.error(`❌ Ошибка: Файл ${inputFile} не найден!`);
    console.error("Положите файл test.docx в папку со скриптом.");
    process.exit(1);
}

console.log("🔄 Начинаю конвертацию...");

// Конвертация
mammoth.convertToHtml({ path: inputFile })
    .then(function(result) {
        // Запись HTML в файл
        fs.writeFileSync(outputFile, result.value);
        
        console.log("✅ Конвертация успешна!");
        console.log(`📄 Результат сохранен в: ${outputFile}`);
        
        // Вывод предупреждений (если есть)
        if (result.messages.length > 0) {
            console.log("\n⚠️ Предупреждения:");
            result.messages.forEach(function(message) {
                console.log("  - " + message);
            });
        }
    })
    .catch(function(err) {
        console.error("❌ Ошибка конвертации:", err);
        process.exit(1);
    });
```

Сохраните файл.

## 5. Создание тестового документа

Теперь нужно создать или использовать тестовый из репозитория файл, который мы будем конвертировать.

1.  Откройте **Microsoft Word** (или LibreOffice Writer / Google Docs).
2.  Создайте простой документ со следующей структурой:
    *   **Заголовок** (используйте стиль "Заголовок 1")
    *   Несколько абзацев обычного текста
    *   **Жирный текст** и *курсив*
    *   Маркированный или нумерованный список
    *   **Таблица** (2x2 ячейки) — чтобы проверить границы
3.  Сохраните файл под именем **`test.docx`**.
4.  Положите этот файл **в ту же папку** `mammoth-test`, где лежит `convert.js`.

**Структура папки должна выглядеть так:**
```
mammoth-test/
├── convert.js
├── test.docx
├── package.json
└── node_modules/
```

## 6. Запуск конвертации

1.  Откройте терминал.
2.  Перейдите в папку проекта (если еще не там):
    ```bash
    cd путь/к/папке/mammoth-test
    ```
3.  Запустите скрипт:
    ```bash
    node convert.js
    ```

---

## 7. Проверка результата

### Если всё прошло успешно:

Вы увидите в терминале:
```
🔄 Начинаю конвертацию...
✅ Конвертация успешна!
📄 Результат сохранен в: .../mammoth-test/test.html
```

В папке появится файл **`test.html`**.

1.  Откройте `test.html` в любом браузере (Chrome, Firefox и т.д.).
2.  Вы должны увидеть ваш текст с заголовками и списками.
3.  **Таблица:** Вы увидите структуру таблицы, но **без границ**.
    *   *Это нормально.* Mammoth извлекает смысл, а не стили. Границы добавляются через CSS на сайте.


## 8. Возможные ошибки и решения

| Ошибка | Причина | Решение |
| :--- | :--- | :--- |
| **`node` не является внутренней или внешней командой** | Node.js не установлен или не добавлен в PATH | Переустановите Node.js, перезапустите терминал |
| **`❌ Ошибка: Файл ... не найден!`** | Файл `test.docx` не в той папке | Положите `test.docx` рядом с `convert.js` |
| **`Cannot find module 'mammoth'`** | Библиотека не установлена | Выполните `npm install mammoth` в папке проекта |
| **Файл `convert.js` не запускается** | Файл сохранен как `convert.js.txt` | Включите отображение расширений в Windows и переименуйте |
| **Крякозябры в выводе** | Проблема кодировки | Сохраните `convert.js` в кодировке **UTF-8** |
